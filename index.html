Dashboard de Activos en Excel

Visualización de activos para empresa retail
Tecnologías: Excel, Tablas Dinámicas
Descripción:

Proyecto realizado como parte de mi puesto como Analista de Cajas, aplicando mis conocimientos en manejo de Excel, consolidación y visualización de datos.

Se desarrolló un dashboard para mostrar a la gerencia de sistemas el avance en la devolución de leasing al proveedor, correspondientes a los activos de cajas (CPU, impresoras, escáneres, monitores, gavetas).

El tablero permitía visualizar anexos asociados, cantidad de equipos, equipos faltantes, ubicación actual, ubicación asignada, porcentaje de avance, entre otros indicadores.

Esto brindó un panorama claro del estado actual de los equipos en leasing por devolver, lo cual facilitó la planificación de acciones para la recuperación de equipos a nivel nacional, centralizada en la sede principal donde se gestionaban los activos, para su posterior empaque y envío al proveedor.

Gracias a esta herramienta, se logró un ahorro tangible al evitar pagos adicionales por equipos cuyo leasing ya hubieran vencido.

Automatización de Reportes

PSAC - Programa en Python para automatizar tareas y reportes
Tecnologías: Python - Pandas - Excel - SQL Server - Linux - Photoshop
Descripción:

Proyecto desarrollado para la automatización de tareas repetitivas en la gestión de servidores Linux y generación de reportes en una empresa de telecomunicaciones.

A través de scripts en Python y Bash (PSAC), se logró verificar automáticamente el estado de los servidores, generar reportes periódicos y realizar modificaciones masivas de archivos promocionales en diferentes entornos.

Pantalla principal del programa PSAC, mostrando el menú de operaciones disponibles.

El sistema permitió reducir tiempos de operación, minimizar errores humanos y facilitar la entrega de información clave a las áreas responsables.

Esta automatización impactó en la eficiencia de los procesos internos y liberó recursos para tareas de mayor valor agregado en la organización.

Análisis Predictivo - Score Crediticio

Modelo de predicción en LightGBM para obtener el score crediticio de emprendedores
Tecnologías: Notebook (Colab) - Visual Studio Code - Python - Arboles de Desicion - Random Forest - LightGBM
Descripción:

Implementación de un modelo predictivo que analiza patrones históricos de consumo y rotación de inventarios para anticipar la demanda, optimizar compras y reducir quiebres de stock.

Permite visualizar escenarios futuros, ajustar estrategias de stock y mejorar la eficiencia operativa en la gestión de inventarios.
